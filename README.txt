Create the visual structure for a project

Utilizing the zipped psd file in this repository create the Schedule an Appointment page as described in the psd file.


The deliverable

Please fork the complete repo and create a folder named 'solution'
in the same folder where you find this READEME.rdoc file.

The folder must contain this structure

  index.html
  index.haml.html
  schedule_appointment.css 

Note that you will be providing both a standard html and a haml version of the markup.

The page should be tested to operate in the following web browsers:
  
  Firefox
  Chrome
  Safari
  Internet Explorer 7+
